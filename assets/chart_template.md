# Markdown 输出模板

八字排盘使用以下 Markdown 格式输出。脚本 `scripts/bazi.py` 已经集成了此格式。

```
# 八字排盘

**出生时间**：YYYY年M月D日 HH:MM
**性别**：男/女
**日主**：X（五行・阴阳）

## 四柱

|     | 年柱 | 月柱 | 日柱 | 时柱 |
|-----|-----|-----|-----|-----|
| **天干** | X | X | X | X |
| **地支** | X | X | X | X |
| **纳音** | XXXX | XXXX | XXXX | XXXX |
| **十神** | XX | XX | 日主 | XX |

## 藏干

- **年柱** X：X（本气）、X（中气）、X（余气） ／ 十神：X→XX、X→XX
- **月柱** X：...
- **日柱** X：...
- **时柱** X：...

## 大运

- **排法**：顺排/逆排
- **起运天数**：N 天
- **起运年龄**：N 岁

| 大运 | 天干 | 地支 | 纳音   | 年龄         |
|------|------|------|--------|--------------|
| 1 | **X** | **X** | XXXX | N–N岁 |
| 2 | **X** | **X** | XXXX | N–N岁 |
| ... | ... | ... | ... | ... |

---
*注：节气日期为近似计算，极端情况可能偏差1日。八字排盘结果仅供参考。*
```

## 输出说明

- **四柱**：年、月、日、时四个柱，每柱包含天干、地支、纳音、十神
- **藏干**：每个地支中隐藏的天干及其强度（本气/中气/余气）和对应的十神
- **大运**：从起运年龄开始，每十年一步大运，共八步（约80年）
- **纳音**：每柱对应的纳音五行，共30种
- **十神**：以日干为中心，其他天干的十神关系

## 注意事项

- 节气计算使用近似公式，日期可能偏差±1天
- 起运年龄精确到小数点后1位
- 年份范围：1900-2100
